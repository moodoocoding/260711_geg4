# 포켓몬 탐험대 

Next.js, TypeScript, Tailwind CSS로 구축된 친근하고 즐거운 분위기의 한 화면짜리 포켓몬 랜딩 페이지입니다.

## 기술 스택
- **프레임워크**: Next.js (App Router)
- **스타일링**: Tailwind CSS v4
- **언어**: TypeScript
- **배포 지원**: Vercel 연동 배포 가능

## 개발 특징
- **반응형 디자인**: 데스크톱에서는 스크롤바 없이 뷰포트에 한 화면으로 들어오는 디자인을 갖추며, 모바일 기기(768px 미만)에서는 세로 스택형 레이아웃으로 변경됩니다.
- **애니메이션**: 
  - 피카츄 이미지의 상하 부드러운 float(둥둥 떠다니기) 애니메이션
  - 첫 진입 시 타이틀 및 문구들의 순차적 fadeIn 페이드인 애니메이션
  - 버튼 및 메뉴 요소들의 자연스러운 마우스 호버(scale-up, hover transitions) 효과
- **로컬 검증**: `npm run build`를 통한 빌드 무오류 검증 완료

## 로컬 실행 방법
```bash
# 의존성 설치
npm install

# 로컬 개발 서버 실행
npm run dev
```
이후 브라우저에서 `http://localhost:3000`으로 접속하여 확인하실 수 있습니다.
